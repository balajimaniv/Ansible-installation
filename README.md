# Ansible-installation
This is a first sample project with ansible installation 

## Install Ansible on Redhat

### commands to execute
    *  rpm -Uvh https://dl.fedoraproject.org/pub/epel/epel-release-latest-7.noarch.rpm

    *  yum -y install ansible
    
    *  ansible --version
